## Como usar

Crea la carpeta de tu nuevo proyecto.

```bash
  mkdir new-project
```
Clona AFA en la carpeta del nuevo proyecto.

```bash
  cd new-project
  git clone https://github.com/dynecore/astro-fir-auth
```

Instala las dependencias.

```bash
  npm install
```

Inicia el servidor

```bash
  npm run start
```
